# Packages Directory

This directory will contain shared packages:

- `@repo/auth` - Authentication
- `@repo/config` - Shared configurations
- `@repo/database` - Prisma ORM
- `@repo/email` - Email templates
- `@repo/i18n` - Internationalization
- `@repo/permissions` - CASL permissions
- `@repo/queue` - BullMQ queues
- `@repo/realtime` - Socket.io
- `@repo/search` - Meilisearch
- `@repo/storage` - S3 + Sharp
- `@repo/types` - TypeScript types
- `@repo/ui` - UI components
- `@repo/utils` - Utilities
- `@repo/validation` - Zod schemas

These will be created in subsequent tasks.

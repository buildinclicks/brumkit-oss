# Tech Stack

## Core

- Next.js 15 (App Router), React 19, TypeScript 5, Node >= 20

## Monorepo

- Turborepo 2.x, pnpm 10.x

## Data

- PostgreSQL 16, Prisma 6.x

## Auth & Permissions

- Auth.js (NextAuth v5)
- CASL for granular authorization

## UI

- Tailwind CSS 4.x
- shadcn/ui + Radix UI

## Editor

- TipTap (free)

## Email

- Resend + React Email

## File Storage & Images

- AWS S3 (prod), MinIO (local), Sharp for resize/thumbnails

## Realtime

- Socket.io (notifications only)

## Background Jobs

- BullMQ + Redis

## Testing

- Vitest, Testing Library, Playwright, MSW

## Notes

- Pin exact versions during implementation; stay on stable releases.
